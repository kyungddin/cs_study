# 컴퓨터 네트워크 하향식 접근 Chapter 1


## 1.1: 인터넷이란 무엇인가?

1. Host vs EndSystem
- Host: 네트워크에서 데이터를 생성하거나 소비하는 장치
    - 즉, 사용자 애플리케이션이 실행되는 장치
- End System: 네트워크의 끝단(edge)에 위치하여 실제로 애플리케이션 데이터를 송수신하는 장치
    - 하지만 실질적으로 Host는 모두 edge에 위치하므로 Host = EndSystem 이라 봐도 무방하다

2. 종단 시스템은 통신 링크와 패킷 스위치의 네트워크로 연결된다
- 통신 링크: 두 네트워크 장치 사이에서 패킷을 전달하는 물리적 또는 무선 연결
    - 동축케이블, 구리선 등의 물리 매체로 구성

3. 각각의 링크들은 다양한 전송률(transmission rate, 링크 대역폭)을 이용하여 데이터를 전송한다
- 전송률은 초당 비트 수를 의미하는 bps(bit per second)를 사용한다

4. 패킷 스위치
- 입력 통신 링크에서 받은 패킷을 출력 통신 링크로 전달하는 장치
    - 즉, 패킷 전달 장치
- 링크 계층 스위치 vs 라우터
    - 링크 계층 스위치(Link-layer Switch): 접속 네트워크에서 사용
    - 라우터(Router): 보통 네트워크 코어에서 사용
    - 접속 네트워크: 종단 시스템 ~ 첫번째 라우터 까지의 네트워크 영역 
        - 쉽게 생각하면 링크 계층 스위치는 End System을 연결하고, 이것이 라우터로 연결되어 접속 네트워크를 구성
        - 링크 계층 스위치는 L2, 라우터는 L3
        - 여담으로 무선AP도 L2다


5. 종단 시스템은 ISP를 통해 인터넷에 접속
- ISP는 패킷 스위치와 통신 링크로 연결된 네트워크

6. RFC
- 인터넷 표준인 IETF의 문서

7. 인터넷 애플리케이션은 종단 시스템에서 수행됨

8. 프로토콜
- 사람의 대화 템플릿을 떠올리면 이해하기 쉽다
- Def: 둘 이상의 통신 개체 간에 교환되는 메시지 포맷, 순서, 메시지 송수신 그리고 그 외의 이벤트에 따른 행동에 대한 정의


## 1.2: 네트워크의 가장자리

1. DSL 인터넷 접속
- DSL(Digital Subscriber Line)
- 일반적으로 가정은 유선 로컬 전화 서비스를 제공하는 Telco로부터 DSL 인터넷 접속 서비스를 받는다
    - 그러나 오늘날 대부분은 케이블 인터넷 접속을 사용한다
    - 특히 한국은 광인터넷인 FTTH를 대부분 사용
- 가정의 DSL 모뎀(MODEM)이 PC의 디지털 신호를 아날로그로 변환
    - Modulation: 디지털 신호를 아날로그 캐리어 신호에 실어서 전달
- PC와 집전화 신호를 분리해주는 스플리터가 존재
- 그렇게 데이터가 중앙국의 DSLAM과 연결
    - DSLAM은 아날로그 신호를 디지털 포맷으로 변환하며, 인터넷과 전화망을 분리한다 (MODEM + 스플리터)

2. 케이블 인터넷 접속
- 케이블 TV 회사의 기존 케이블 TV 인프라스트럭처를 이용한 인터넷 접속
- 전통적인 동축케이블을 사용한다
    - 그렇기에 케이블 TV와 신호를 분리시켜줄 필요가 없으며, 스플리터도 없다
    - 수백 가구의 인터넷이 동축케이블로 연결되어 광 노드로 이어진다
    - 그리고 광 노드는 CMTS와 광케이블로 연결되어있다
